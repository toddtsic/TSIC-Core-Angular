Schedule Hub

This is where you build and manage your event's game schedule. The hub has four modes across the top, and you generally move through them left to right — Configure what the schedule needs, Schedule the games, check the Master view, and run QA before you publish.

Use a desktop.

The grid, drag-and-drop, and game placement need a full-size screen — the hub asks for a desktop browser rather than a phone or small tablet.

The four modes

Configure
Set what the schedule is built from — game dates, fields, timeslots, pairings, and pools. Start here on a new event.
Schedule
Build games automatically, then place any championship/bracket games onto the grid by hand.
Master
The whole event on one grid — the view everyone else consumes.
QA
Checks for conflicts — a team double-booked, a field clash — before you publish.

Configure first — it's the part that's easy to skip

A schedule can only be built once the event knows its dates, its fields, the timeslots on those fields, and the pairings (who plays whom) for each team count. On a brand-new event the Configure tab pulses "Start here" for exactly this reason. When an age group needs pools, set those under Pools; playoff seeding lives under Bracket Seeds.

If you're running a new season of a recurring event, the hub tries to carry your prior year's field and date setup forward automatically — but always confirm it in Configure before you build.

Scope: Event, Age Group, or Division

The navigator on the left lets you work at three levels. Build and delete always act on the scope you've selected:

Event → Age Group → Division

Drill down to a single division to fine-tune it; step back up to Event to build everything at once. The Build button always names the current scope, and reads Re-Build once that scope already has games.

Building the games

  • In Schedule mode, click Build (or Re-Build) to auto-generate round-robin games for the selected scope. A setup check runs first and lists anything still missing.
  • Championship / bracket games are placed by hand. After the round-robin build, the hub switches you to the affected division and asks you to drop each championship game onto an open slot — use Add Row to create a timeslot at the right interval if you need one.
  • Two placement styles: mouse (click a game, click an open slot) or keyboard (Rapid Placement — pick a field and time by typing). Your choice is remembered.

Common mistakes

  • Building before configuring. If the setup check blocks you, finish Configure first — missing dates, fields, timeslots, or pairings are the usual culprits.
  • Building at the wrong scope. Check the scope label on the Build button — it's easy to rebuild one division when you meant the whole event, or vice-versa.
  • Walking away with championship games unplaced. Round-robin games build automatically; bracket games don't. If the hub shows a championship banner, you still have games to place.

If you get stuck

Run QA before publishing — it surfaces conflicts you can't easily eyeball. If a build won't start, the setup-incomplete panel tells you exactly what's missing and where to fix it.
