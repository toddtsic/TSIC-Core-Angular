Open a Registration

After you sign in, this screen lists everything on your account so you can pick which one to open. One login can hold several registrations — different children, teams, or roles across events — so this is where you choose the one you want to work in right now.

Choosing a registration

Each row is one registration. Click it to open that registration and continue.

  • The bold line is the event; the muted line below identifies exactly which one it is — for a player, that's the name, age group, and team. That detail line is how you tell two children (or two teams) apart.
  • Registrations are grouped by role — for example a Player section and a Club Rep section — each shown under its own heading.
  • If a role has a lot of registrations, it appears as a search box instead of a list — just start typing to filter.

Switching accounts

The Logout button at the bottom signs you out — use it if you reached the wrong account and need to sign in as someone else.

Tip: even if you see only one registration, you still tap it to open it — the screen never opens one for you, so you always know which you're in.
