Job Settings

Each tab saves separately.

The floating Save button saves only the tab you're on. A dot on a tab means it still has unsaved changes — save each dotted tab before you leave.

I changed a setting, switched tabs, and my change was gone. Why?

Each tab saves on its own. When you switch tabs with unsaved changes, you're asked whether to discard them — choosing to move on drops them. Press Save first (or answer Stay at the prompt) to keep your changes.

Where's the Save button?

It only appears when the tab you're on has unsaved changes — a floating Save button at the bottom-right. It saves that one tab, so if several tabs have dots, save each of them.

The "Allow ... Registration" toggle is greyed out. How do I turn registration on or off?

Those switches — player, team, and staff/coach registration, plus public schedule and roster access — are managed in Quick Links. They show here read-only so you can see the current state without two screens disagreeing. Change them in Quick Links and they update here.

What does the padlock icon mean?

A padlock marks a SuperUser-only field. Directors can see it but can't change it — things like processing fees, sport / customer / billing type, recurring billing, and store settings.

Why can't I turn on eCheck?

eCheck rides on check payments, so Check must be an allowed method first — set Allowed Methods to CC or Check or Check Only, then the Enable eCheck box unlocks. Note the processing and eCheck fees themselves are SuperUser-only.

Where did the "full payment required" setting go?

It's no longer a single job-wide switch. Full-payment phase is now set per league, age group, or team on the fee cards (the most specific scope wins), so different divisions can differ. Existing job-level values are still honored as the fallback.

On the Adult tab, what does "Apply coach form change" do?

Picking a different coach form and applying it rebuilds the form from that template — it's a structural swap, not a field tweak. To edit individual fields on the existing form, use Profile Editor → Adult instead. “Require USA Lacrosse #” is only available on forms that support it.

How do invitation tokens work?

When Require Invitation Token is on (Players or Teams tab), registrants need a valid invite link to start. Generate those links in batch emails with !INVITE_LINK for players and !CLUBREP_INVITE_LINK for club reps. With the token off, registration is open to anyone.

What size should banner images be?

Upload the banner background at 1920 × 422 (50:11) so it fits without cropping. The overlay image is resized to max 800px wide and the header logo to max 400px wide. If “Use custom banner images” is off, a default TSIC banner shows instead.

How do I add several dropdown options at once?

On the Dropdowns tab, type multiple values separated by semicolons in the add box and they're added in bulk. These values feed the dropdowns on the player and team registration forms.

What are player and team donations?

Enabling them (Payment tab) adds an optional donation field to the payment page. Any amount entered is charged in full, plus processing, together with the registrant's payment.

Tip: setting up a new event? Walk the tabs left to right — General, Branding, Payment, then the registration tabs — saving as you go. The dots tell you which tabs you've touched but not yet saved.
