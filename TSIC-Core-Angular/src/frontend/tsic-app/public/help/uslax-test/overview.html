US Lax Membership Lookup

A quick spot-check for a single USA Lacrosse membership number. Type a number, verify it, and see whether it's active and whether it's valid long enough for this event. Use it when a family disputes a membership problem, or to sanity-check a number before you rely on it.

How to use it

  1. Enter a USA Lacrosse membership number (6–12 digits) and click Verify (or press Enter).
  2. The result shows the member's status and expiration date straight from USA Lacrosse.
  3. It's compared against this event's valid-through date — the date a membership must remain active past. If the membership expires before then, it's flagged as not valid for the event.
"Active" isn't the whole story.

A membership can be active today but still expire before your event's valid-through date. This tool checks both, so you catch the ones that lapse mid-season.

What it is (and isn't)

This is a read-only lookup — it doesn't change a registrant's record or fix anything. To reconcile memberships across your whole roster or email people about missing ones, use the US Lax Membership tool instead.

Common mistakes

  • Mistyping the number. A single wrong digit returns the wrong person or no result — copy it exactly.
  • Reading "Active" as "good for the event." Check the expiration against the event's valid-through date, which this page shows for you.
  • Expecting it to update the registrant. It's a diagnostic; fixing the record happens elsewhere.
