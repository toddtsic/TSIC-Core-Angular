US Lax Membership Lookup

This checks one number at a time.

It's a diagnostic lookup, not a bulk reconciler. For the whole roster, use the US Lax Membership tool.

What counts as a valid membership number?

A USA Lacrosse number of 6 to 12 digits. If you enter letters or the wrong length, the tool tells you before it even looks it up.

The membership is "Active" but flagged as not valid — why?

Because it expires before this event's valid-through date. Active means valid right now; the event needs it to stay active through a later date. A membership that lapses mid-season is active today but still won't cover the event.

What's the "valid-through" date it compares against?

It's a setting on the event — the date a membership must remain active past to satisfy this event's requirement. The page shows it so you know what the expiration is being measured against. If it reads "Not set," the event hasn't defined one.

I got "service temporarily unavailable." What now?

The lookup calls USA Lacrosse live, so a hiccup there stops the check. Wait a moment and try again — it's not a problem with the number you entered.

Does looking a number up change the registrant's record?

No. This page only reads and displays — it never writes anything back. Updating a registrant's membership status is done in the registrant's record or the US Lax Membership tool.

Tip: keep this open when a parent calls about a membership dispute — you can read them the exact status and expiration on file with USA Lacrosse in seconds.
