View Schedule

This is the admin's window into the whole event's games — browse the schedule, check standings and brackets, find team contacts, and (as an admin) enter or fix scores. It's the same schedule the public sees, plus the editing tools only you get.

Four tabs

Games
Every scheduled game. Enter or edit scores here, and open a game to adjust its details.
Standings
Pool standings, updated from the scores you enter — the source for bracket seeding.
Brackets
Playoff brackets as they fill in from results.
Contacts
Team contacts, for when you need to reach a coach or club rep.

Narrowing the list with filters

A full event has hundreds of games, so filtering is how you get to the ones you want. You can filter two ways:

By Club

Drill Club → Age Group → Division → Team — best when you're looking at one club's games.

By League

Drill League → Age Group → Division → Team — best for working within the event's structure.

Add day, time, and field filters, or flip on Unscored to show only games still missing a result. Each filter you pick becomes a removable chip, so you can always see what's applied and clear it.

Entering scores (admin)

  • On the Games tab, enter a game's score directly — standings and brackets update from it.
  • Use the Unscored filter to work through exactly the games that still need results — faster than hunting for gaps.
  • Open a game to edit fuller details or run the game clock where that's in play.

Common mistakes

  • Not seeing games you expect. A filter is probably still applied — check the chips and clear the ones you don't want.
  • Wondering why standings look off. Standings are computed from entered scores; a missing or wrong score is usually the cause. The Unscored filter finds the gaps.
  • Confusing this with the Rescheduler. Here you record results and tweak game details; to move a game to a different field or time, use the Rescheduler.
