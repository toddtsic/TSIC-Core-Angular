Bracket Seeds

You assign a pool and a rank — not a team.

The actual team is resolved from final standings. That's the one idea that makes this whole screen click.

Why can't I pick the actual teams for a playoff game?

Because at seeding time you usually don't know them yet — pool play isn't finished. You set a Seed Pool and a Seed Rank (e.g. "Pool A, Seed 1"), and once standings are final the bracket automatically pulls in whichever team earned that spot.

The page says there are no bracket games. Where are they?

Bracket/playoff games are created during scheduling, as non-round-robin game types (championship, final, semi, quarter). This screen only seeds games that already exist. If the list is empty, build the playoff structure in the Schedule Hub first, then come back to seed it.

What does "Pool A #1 plays Pool B #2" actually mean?

Each game has two sides. One side is set to Pool A, rank 1; the other to Pool B, rank 2. So the winner of Pool A meets the runner-up of Pool B. You're describing the matchup by finishing position, and the teams drop in from standings.

How do I know I've seeded it correctly?

Use the Standings panel on this page. It shows each pool's order — record, points, goal differential. Cross-check that the seeds you assigned match the way you expect the pools to finish, so no slot is left pointing at a rank that pool can't produce.

Can I change a seed after games have been played?

You can, but be deliberate — a seed drives who advances into that game. After any change, re-check the affected bracket games so a downstream matchup doesn't end up pointing at the wrong pool or rank.

This event says "Re-seed mode." What's different?

On a re-seed event, a seed pool can be drawn from any round-robin age group across the job, not just this game's own age group — and each side's available ranks are capped at the size of the pool you choose. The banner at the top flags when you're in that mode.

Tip: seed the bracket before pool play ends. Because slots resolve from standings, a fully-seeded bracket simply fills itself in the moment the last pool game is scored — no scramble on finals morning.
