Edit Profile — questions

Which fields can I actually change?

Your email, phone, and mailing address (street, city, state, ZIP). Choose Save Changes when you're done. The other fields are shown for reference but greyed out.

Why can't I edit my name?

Your name is locked so an account can't be handed to a different person just by renaming it. If your name genuinely needs correcting, contact the event director and they'll take care of it.

How do I change my password or username?

Not from this page — username and password are locked here. Use the sign-in screen's password-reset link, or contact the event director if you're stuck.

My club name is wrong.

Club name can't be edited here. Ask the event director to update it — it's tied to how your teams are grouped, so it's changed on their side.

I clicked Save but nothing happened.

Check for a field marked in red — an invalid email or ZIP will block the save until it's fixed. Once everything's valid, Save Changes confirms and returns you to the home page.

The page won't load my profile.

If you see "Unable to load your profile," refresh and try again. If it keeps failing, sign out and back in, or contact the event director.
