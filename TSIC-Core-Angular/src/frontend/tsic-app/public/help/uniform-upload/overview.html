Upload Uniform Numbers

This tool sets uniform numbers (and day groups) for a whole event at once, using Excel instead of editing players one by one. You download a spreadsheet pre-filled with every registered player, type in their numbers, and upload it back — the system applies the changes in bulk.

How it works — two steps

  1. Download Template — an Excel (.xlsx) file with a row for every player in this event. Fill in the UniformNo and DayGroup columns. Leave the other columns (including the ID column) exactly as they are.
  2. Upload File — drop the saved .xlsx back in (or click to browse) and choose Upload & Apply. Only .xlsx files up to 10 MB are accepted.

Reading the results

After an upload you get a summary:

  • Updated — rows whose number/group changed.
  • Unchanged — rows that already matched, so nothing to do.
  • Errors — rows that couldn't be applied, listed below with the row number and a reason.
  • Total Rows — everything the file contained.

If there are errors, fix those rows in the spreadsheet and re-upload — re-uploading is safe and just re-applies.

Don't touch the ID column.

It's how the system knows which player each row belongs to. Change or delete it and those rows land in the Errors list.

Common mistakes

  • Uploading a different file format. It must be the .xlsx template — not a CSV, not an old .xls, not a re-saved copy that changed the columns.
  • Editing or reordering columns. Only change UniformNo and DayGroup; leave the structure alone or the upload can't map rows.
  • Starting from a stale template. Download a fresh template if your roster changed — an old file won't include newly-registered players.
