Referee Assignment

Import your officials before you assign.

The grid can only place referees who are already in your pool — load them first via CSV.

How do I add my referees?

Use the CSV File import to load your whole list of officials at once. After importing, they're available to assign to any game on the grid.

How is the grid laid out?

Fields run across the top and time slots down the side, so each cell is the game at that field and time. Use the day, time, age-group, and field filters to narrow it to the games you're staffing right now.

Do I have to assign the same crew game by game?

No. Copy a crew from one game to others, or bulk-assign a referee down a field's day by setting a Number of Timeslots, a Skip Interval, and a Direction — that places them across consecutive games in one action.

How do I avoid double-booking a referee?

Watch the grid as you assign — a referee placed on two games in the same timeslot is a conflict. Bulk assignment and crew-copy are quick, so scan the affected column afterward to confirm no one landed on overlapping games.

Where can I see each referee's own schedule?

Use the Referee Calendar for a per-official, game-by-game view of what each one is assigned. This screen is for making assignments; the calendar is the readable schedule of the result.

Tip: staff one field's full day at a time. Filtering to a single field makes gaps obvious and lets you assign a crew straight down its timeslots without hopping around the grid.
