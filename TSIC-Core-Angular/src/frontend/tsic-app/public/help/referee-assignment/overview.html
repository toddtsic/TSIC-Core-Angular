Referee Assignment

Assign officials to games. The screen is a field × time grid — fields across the top, time slots down the side — so every game sits where it's played. Pick a game and assign the referee(s) it needs.

Build your referee pool first.

You can only assign refs you've got. Import your list of officials before you start filling the grid.

Getting your referees in

Use the CSV File import to load your officials in one go, rather than adding them one at a time. Once imported, they're available to assign to any game.

Assigning

  • Filter by Game Days, Game Times, Age Groups, and Fields to focus on the slice you're staffing.
  • Open a game and Assign the referee(s) it needs.
  • Copy a crew from one game to others so you don't re-enter the same officials repeatedly.
  • Bulk-assign a ref down a field's day — set the Number of Timeslots, a Skip Interval, and a Direction to place them across consecutive games at once.

Avoiding conflicts

A referee can't be in two places at once. When you assign across a field or copy a crew, keep an eye on the grid so the same official isn't landed on two games in the same timeslot.

Common mistakes

  • Assigning before importing. With no referees loaded, there's no one to place — import first.
  • Double-booking an official. Bulk-assign and crew-copy are fast, but check they didn't put a ref on overlapping games.
  • Leaving games uncovered. Filter to a field's whole day and scan for empty games so none go unstaffed.
