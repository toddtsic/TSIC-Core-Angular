Team Links

Team Links let you post handy web links on your event's player-facing pages — things like a team's practice schedule, a video channel, a fundraising page, or a shared photo album. Each link can be attached to one team or shown to all teams, so families see exactly the links that apply to them.

What a link holds

  • Team — the team the link belongs to, or — All Teams — to show it everywhere.
  • Label — the clickable text families see (keep it short and clear, e.g. "Practice Schedule").
  • URL — the full web address, starting with https://.

Adding and managing links

  1. Click Add, pick a team (or All Teams), enter a label and the URL, and save.
  2. Use Edit on any row to fix a label or a broken URL.
  3. Use Delete to remove a link that's no longer relevant — it disappears from the player pages.
Paste the whole URL.

Include https:// — a bare address like myteam.com/schedule may not open correctly for families.

Common mistakes

  • A partial URL. Without the https:// prefix the link can break. Copy the full address from your browser's address bar.
  • Assigning to the wrong team. A link set to one team only shows on that team's pages — double-check the Team field, or use All Teams for event-wide links.
  • Cryptic labels. "Click here" tells families nothing; name the destination so they know what they're opening.
