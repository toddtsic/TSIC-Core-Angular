Mobile Scorers

These accounts are logins for the mobile scoring app.

Hand one to each person entering scores; results they post flow straight into your schedule.

What can a scorer account actually do?

Enter game scores from the mobile app — and that's it. A scorer login doesn't open the admin site or any other tools, so it's safe to hand out to volunteers and field marshals.

How do I give someone scoring access?

Add a scorer to create a login, then share those credentials with the person who'll enter scores. They sign into the mobile scoring app with it. Create separate accounts where you can, so it's clear who posted what.

How do I revoke access without deleting the account?

Switch the account to inactive. It keeps the record but stops the login from working — handy at the end of an event, or if you want to pause someone temporarily. Delete only when you're sure you won't reuse it.

Where do the scores they enter show up?

In the same place as scores you'd type yourself — the schedule. Standings and brackets update from them, so a scorer keeping up on their phone keeps the whole event's results live.

I don't see this feature. Why?

Mobile scoring is a per-event capability. If it's not turned on for this job, the scorer tools won't appear. A SuperUser can confirm whether the mobile feature is enabled for the event.

Tip: set up your scorer accounts a day or two before the event and test one login in the app. Sorting out credentials on Saturday morning is the last thing you want while games are starting.
