Get Reconciliation Records

Runs the full month-end close: pulls last month's Authorize.Net settled batches (registration and merch) into Txs, then downloads a .zip with two separate QuickBooks IIF files — reg-consolodated.iif and merch-consolodated.iif — plus their backing Excel. No more manual Excel-to-IIF step. Idempotent — re-running for the same month wipes that month's Txs rows then repopulates from a fresh ADN pull. The pull always hits production ADN (even when running on dev) so the dev Txs table mirrors prod for debugging.
