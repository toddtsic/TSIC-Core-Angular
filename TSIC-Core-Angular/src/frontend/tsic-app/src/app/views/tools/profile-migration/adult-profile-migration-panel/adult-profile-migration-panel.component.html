About Adult Migration

This materializes the role-keyed adult form (Coach/Staff, Referee, Recruiter) into AdultProfileMetadataJson for every job on a canonical profile. The two profiles — AC1 (Standard) and AC2 (Apparel) — are mapped from each job's legacy RegformName_Coach.

USA Lacrosse is a per-job capability, not a separate profile: jobs whose legacy form required it get a required sportAssnId field prepended to the coach block. The USLax count below is how many of a profile's jobs carry that capability.
