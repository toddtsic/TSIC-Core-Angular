Rankings System Guide

What This System Does

This automated system matches your registered tournament teams with usclublax.com national rankings, providing accurate ranking data to make informed seeding decisions for your tournament.

Matching Process

The system uses a multi-factor comparison with noise-word filtering, state awareness, and strict binary validations:

Factor Type How It Works
Club Name Critical Fuzzy match after stripping noise words (Select, Premier, etc.)
Color Binary Gate Both must have same color, or neither has a color
Year/Level Binary Gate Graduation years must match (supports HS level keywords)
State Penalty Different states apply -20% penalty (not a hard rejection)
Manual Override User Manually match unmatched teams via the Match button
Binary Validations

Teams are automatically rejected if:

  • Different Colors: "Elite Red (2026)" will NEVER match "Elite Blue (2026)"
  • Different Years: "Elite Red (2026)" will NEVER match "Elite Red (2027)"
  • Color mismatch: One has a color and the other doesn't

Use the Match button on unmatched teams to manually override when the algorithm misses.

Confidence Levels
75%+ High — Very likely correct
50-74% Medium — Review recommended
Manual User-assigned match
Workflow
  • Step 1: Select age groups and click Find Matches — the system proposes matches but nothing is saved yet
  • Step 2: Review the matches — you can override any match or manually assign unmatched teams using the Match buttons
  • Step 3: Switch to Save to Teams — choose a confidence threshold and decide which matches to keep. Only you make the call to commit
  • Step 4: Ranking data appears in the Nat'l Rank column in Pool Assignment for seeding decisions
