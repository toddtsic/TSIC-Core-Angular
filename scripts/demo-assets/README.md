# Demo Branding Assets

Place stock demo images here. The `Clone-DemoDb.ps1` script copies these
over every job's branding images during the morph process.

## Required Files

| Filename | Purpose | Recommended Size |
|----------|---------|-----------------|
| `demo-banner-bg.jpg` | Hero banner background | 1920 x 600px |
| `demo-banner-overlay.png` | Banner overlay logo/graphic | 800 x 400px (transparent PNG) |
| `demo-logo-header.png` | Header logo | 400 x 100px (transparent PNG) |

## Tips

- Use generic sports imagery (stock photos, abstract graphics)
- Avoid anything that identifies a real organization
- Keep files under 1MB each for fast cloning
- Multiple sets can be added later (demo-banner-bg-soccer.jpg, etc.)
